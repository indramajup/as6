Code Geass

Code Geass (ญี่ปุ่น: コードギアス) หรือชื่อเต็ม Code Geass: Lelouch of the Rebellion เป็นอนิเมะแนวแอ็กชัน–การเมือง–ไซไฟ

เล่าเรื่องของ Lelouch Lamperouge เจ้าชายที่ถูกเนรเทศจากจักรวรรดิ Holy Britannian Empire เขาได้รับพลังลึกลับชื่อ “Geass” ที่สามารถสั่งให้ใครก็ตามทำตามคำสั่งได้หนึ่งครั้ง

เลลูชจึงสวมหน้ากากในนาม “Zero” นำกองกำลังกบฏเพื่อล้มล้างจักรวรรดิ และสร้างโลกที่ดีกว่าสำหรับน้องสาวของเขา

สรุปสั้นๆ: เป็นเรื่องการปฏิวัติ การวางแผนเชิงกลยุทธ์ และศีลธรรมของการใช้พลังอำนาจ ✨

Lelouch Lamperouge

อัจฉริยะผู้ใช้สติปัญญาเป็นอาวุธ เขาสวมหน้ากากในนาม “Zero” และวางแผนโค่นล้มจักรวรรดิด้วยกลยุทธ์ที่เหนือชั้น

The Power of Geass

พลังลึกลับที่สามารถบังคับให้ผู้อื่นทำตามคำสั่งได้หนึ่งครั้ง พลังนี้คือกุญแจสำคัญของการปฏิวัติ

The Rebellion

การลุกขึ้นสู้ของ Black Knights คือจุดเริ่มต้นของสงคราม และการต่อสู้เพื่อเสรีภาพ

Moral Conflict

เรื่องราวสะท้อนคำถามว่า “อำนาจเพื่อความถูกต้อง” คุ้มค่ากับการเสียสละหรือไม่
